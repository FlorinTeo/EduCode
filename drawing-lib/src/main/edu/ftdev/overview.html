Drawing Library

The Drawing Library provides tools for creating, rendering, and manipulating graphical elements. It includes support for:

  • Drawing and rendering shapes, images, and maps.
  • Interactive features like zooming, panning, and user input handling.
  • Generating STL models for 3D printing.
  • Customizable components for specific use cases like Sudoku boards and Moon maps.

This library is available here: drawing-lib-2.0-shaded.jar
